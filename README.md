# Student Grade Predictor

## Overview

This project uses machine learning to predict students' final grades based on their attendance, assignment scores, midterm marks, and study hours.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Google Colab

## Features

* Generates a student performance dataset
* Trains a Linear Regression model
* Predicts final grades for new students
* Visualizes actual vs predicted grades

## Dataset Attributes

* Attendance
* Assignment_Score
* Midterm_Score
* Study_Hours
* Final_Grade

## Machine Learning Algorithm

* Linear Regression

## Results

The model achieved a high R² score, demonstrating strong prediction accuracy.
